Indigo Minimalist Jekyll Template - Demo ·

This is a simple and minimalist template for Jekyll for those who likes to eat noodles.

What has inside? | Setup? | Settings | How to | Tests | Problems

What has inside?

  • Gulp, BrowserSync, Stylus ~RSCSS~ and SVG
  • Tests with Travis
  • Google Speed: 98/100
  • No JS.

Setup

  1. to the project.
  2. Install Jekyll, NodeJS and Bundler.
  3. Fork the project Indigo
  4. Edit _config.yml with your data.
  5. bundle install
  6. npm install && npm install -g gulp
  7. gulp
  8. open in your browser: http://localhost:3000

Arti Annaswamy wrote a really nice tutorial of "how to install jekyll". Here's the part 1 and part 2.

Settings

You must fill some informations on _config.yml to customize your site.

name: John Doe
bio: 'A Man who travels the world eating noodles'
picture: 'assets/images/profile.jpg'
...

and lot of other options, like width, projects, pages, read-time, tags, relateds, animations, etc.
